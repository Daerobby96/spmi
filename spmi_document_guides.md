# 📑 Panduan Kelengkapan Dokumen SPMI

Dokumen-dokumen di bawah ini adalah kelengkapan standar yang harus ada dalam implementasi Sistem Penjaminan Mutu Internal (SPMI) Perguruan Tinggi untuk memenuhi kriteria Akreditasi (BAN-PT/LAM).

## 1. Daftar Dokumen yang Harus Disediakan

| Kategori | Nama Dokumen | Fungsi Utama |
| :--- | :--- | :--- |
| **Pilar Utama (Induk)** | Kebijakan SPMI | Landasan filosofis dan arah mutu institusi. |
| | Manual SPMI | Panduan teknis pelaksanaan siklus PPEPP. |
| | Standar SPMI | Pernyataan standar (kriteria) yang harus dicapai. |
| | Formulir SPMI | Template/form rekaman kegiatan. |
| **Operasional** | SOP (Standar Operasional Prosedur) | Alur kerja teknis setiap proses di unit. |
| | Instruksi Kerja (IK) | Panduan detail penggunaan alat/sistem. |
| | Dokumen Tupoksi | Deskripsi tugas fungsional dan struktural. |
| **Evaluasi & Audit** | Rencana Akreditasi/Audit | Jadwal pelaksanaan pemantauan mutu. |
| | Instrumen Audit | Daftar periksa untuk menilai ketercapaian standar. |
| | Laporan Hasil Audit (LHA) | Ringkasan temuan ketidaksesuaian. |
| **Peningkatan** | Rencana Tindak Lanjut (RTL) | Langkah strategis perbaikan dari pimpinan. |
| | Notulensi RTM | Bukti rapat tinjauan manajemen. |

## 2. Petunjuk Unggah di Sistem SPMI

Berikut adalah panduan navigasi ke mana Anda harus mengunggah dokumen-dokumen di atas di dalam aplikasi:

### A. Menu: Dokumen Mutu (Manajemen Dokumen)
**Gunakan bagian ini untuk dokumen yang bersifat sebagai "Panduan" atau "Referensi Tetap".**
*   **Dokumen yang diunggah di sini:** Kebijakan, Manual, Standar, SOP, IK, dan Formulir kosong.
*   **Cara:** Masuk ke menu **Dokumen Mutu** > **Tambah Dokumen**. Pastikan memilih kategori yang sesuai (misal: Kategori "SOP" atau "Standar").

### B. Menu: Audit Mutu Internal (Manaksanaan Audit)
**Gunakan bagian ini untuk bukti audit dan tindak lanjut temuan.**
*   **Dokumen yang diunggah di sini:** Rencana Audit, Instrumen Audit, dan Bukti-Bukti Temuan.
*   **Cara:** 
    1. Klik detail Audit pada menu **Audit**.
    2. Unggah file pendukung di bagian **Temuan** atau saat Auditee mengisi **Tindak Lanjut**.

### C. Menu: Monitoring & Evaluasi
**Gunakan bagian ini untuk bukti capaian indikator (Evidence).**
*   **Dokumen yang diunggah di sini:** RPS, Bukti Foto Kegiatan, Laporan Kinerja Bulanan, Data Sertifikasi, dll.
*   **Cara:** Masuk ke menu **Monitoring**, cari Indikator yang relevan, lalu klik **Input Capaian** dan unggah file buktinya di sana.

### D. Menu: Laporan
**Gunakan bagian ini untuk mengunduh kompilasi.**
*   Sistem akan secara otomatis merangkum dokumen-dokumen di atas menjadi laporan PDF atau Excel yang siap digunakan untuk borang akreditasi.
